# Build Photos

This folder contains photos documenting the build process and final assembly.

## Photo Guide

### Suggested Photos to Add:

1. **01_frame_assembly.jpg** - Laser-cut frame parts before and after sanding
2. **02_electronics_layout.jpg** - Component placement and wiring layout
3. **03_led_installation.jpg** - NeoPixel strip installation
4. **04_flapper_mechanism.jpg** - 3D printed flapper arm and servo mounting
5. **05_completed_top.jpg** - Fully assembled robot from above
6. **06_completed_side.jpg** - Side view showing ground clearance
7. **07_remote_control.jpg** - TTGO T-Display with joystick and buttons
8. **08_robot_action.jpg** - Robot in motion with LEDs active
9. **09_motor_couplers.jpg** - 3D printed motor-to-wheel couplers installed
10. **10_wiring_detail.jpg** - Clean wiring and cable management
11. **11_battery_mount.jpg** - Battery installation and connections
12. **12_comparison.jpg** - Size comparison with common object

### Tips for Good Build Photos:
- Use good lighting (natural daylight is best)
- Clean background (plain surface or mat)
- Multiple angles for complex assemblies
- Include scale references where helpful
- Close-ups for detailed components
- Action shots for movement demonstrations

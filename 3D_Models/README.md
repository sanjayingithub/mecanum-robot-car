# 3D Models Directory

This folder contains STL files for custom 3D printed components used in the mecanum robot car build.

## Files

### motor_coupler.stl
- Motor shaft to wheel hub coupler
- Print with 80% infill for strength
- Material: PLA or PETG

### flapper_arm.stl
- Servo-mounted flapper mechanism for ball kicking
- Print with 80% infill
- Material: PETG recommended (more impact resistant)

## Print Settings

**Recommended:**
- Layer Height: 0.2mm
- Infill: See individual file notes
- Supports: As needed
- Raft/Brim: Use for parts with small contact area

**Notes:**
- All files designed for standard FDM printers
- No special hardware required
- Test fit before full assembly
- Sand contact surfaces for smooth fit
